## Installation

### Add LDRX code into HTML page

Download LDRX: [ldrx.min.js](https://github.com/AntoineLandrieux/LDRX/releases/tag/Interpreter)

```html
<!DOCTYPE html>
<html>
<head>
    ...
    <script src="ldrx.min.js">
</head>
<body>
    ...
    <ldrx-run>
        \\ Code
        print "<p>This website use LDRX.</p>"
    </ldrx-run>
</body>
</html>
```
